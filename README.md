# Projeto de Extensão: Currículo Acessível
1. O Problema
No cenário atual do mercado de trabalho, o currículo é a porta de entrada para qualquer oportunidade. No entanto, uma parcela significativa da população local — composta por trabalhadores informais, jovens em busca do primeiro emprego e pessoas com baixa fluência digital — enfrenta barreiras invisíveis. A dificuldade em formatar um documento no Word, organizar as experiências de forma profissional ou até o custo de pagar por uma impressão em lan house acaba excluindo esses cidadãos de processos seletivos.

2. A Solução
O projeto consiste no desenvolvimento de uma plataforma web simplificada, focada em UX (Experiência do Usuário) inclusiva. O objetivo é que qualquer pessoa, mesmo com conhecimento básico de internet, consiga:

Preencher seus dados através de um formulário intuitivo.

Gerar automaticamente um arquivo PDF padronizado e profissional.

Acessar a ferramenta gratuitamente via celular ou computador.

3. Diferenciais do Projeto
Diferente de geradores de currículo complexos e cheios de anúncios, este site foi pensado para a realidade da nossa cidade:

Linguagem Simples: Sem termos técnicos que confundam o usuário.

Foco na Agilidade: Menos cliques, mais resultado.

Leveza: Otimizado para funcionar bem mesmo em conexões de internet instáveis ou planos de dados limitados.
